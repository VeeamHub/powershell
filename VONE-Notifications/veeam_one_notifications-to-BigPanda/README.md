# Veeam ONE Notifications to BigPanda

## Author

Adam Congdon

## Function

This script takes parameters from Veeam ONE alerting and sends alerts to BigPanda via REST API.

It includes debug logging to a file for troubleshooting.

## Known Issues

* *None*

## Requirements

* Veeam ONE 12.2
  * *might work with other versions but untested*

## Usage

```powershell
Get-Help .\VeeamONE-Notification-BigPanda.ps1 -Full
```
